Hello! Welcome to Kaleidoscope!

Kaleidoscope is an interactive experience designed to immerse the user in a new environment. In order participate, there are a few things to know:

  1. For an optimal experience, please wear headphones.
  2. There are two modes: edit and view. In edit, you may create experiences dragging icons onto the image. In view, you can fully experience the environment without any distractions.
  3. If you ever get confused or want to start over, simply refresh to view the directions once again.
